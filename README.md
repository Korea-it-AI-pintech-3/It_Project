## 기능 명세서

### 관리자 기능
1. 로그인
   - 회원가입
   - 회원탈퇴
   - 로그인 데이터 조회
   - 아이디 및 비번 수정
2. 게임(랜덤 가위바위보)
   - 컴퓨터
      - 가위, 바위, 보 중 택 1 제시
   - 플레이어
      - 게임 로그
      - 게임 전적
      - 게임 시간
   - 버그
      - 예기치 않은 오류 검사
3. 종료
   - 게임 다시 시작
   - 게임 재접속
   - 프로그램 종료
### 사용자 기능
1. 로그인
   - 아이디 & 비밀번호 설정
   - 아이디 & 비밀번호 찾기
   - 이메일 주소 기억하기
2. 게임화면
   - 마이페이지
      - 게임 로그 조회
      - 승률 조회
      - 회원정보(닉네임) 수정
   - 게임 시작
      - 랜덤 가위바위보
         -  무작위 가위 바위 보 5개 제공
         - 제공된 가위 바위 보 중 택 1
         - 컴퓨터와 가위바위보 후 사용한 카드 제거
         - 1게임당 5회 플레이 OR 3선승 시 종료
      - 가위바위보
      - 종료
